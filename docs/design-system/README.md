# Loopin Design System

> Linear·GitHub·Raycast처럼 **구조적이고, 의도적이며, 기술적으로 깔끔한** 인터페이스.

## 핵심 원칙 3가지

| 원칙            | 한 줄 요약                                               |
| --------------- | -------------------------------------------------------- |
| **No-Line**     | 명시적 선(`border`) 대신 surface-container 계층으로 구분 |
| **Tonal Depth** | 5단계 surface-container로 시각 계층 표현                 |
| **Calm Motion** | easing-standard + 360ms 이하, 바운스 금지                |

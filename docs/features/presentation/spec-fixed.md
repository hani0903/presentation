# 홈 페이지 기능 정의서 (확정)

## 기능 개요

사용자가 서비스에 처음 진입했을 때 서비스의 목적을 이해하고, 퀴즈 풀이를 시작할 수 있는 홈 페이지를 제공한다. 모바일 퍼스트로 구현하며, 앱 메인 화면 느낌의 UI를 목표로 한다.

## 기능 요구사항

### 헤더 영역

- 서비스 이름 `Loopin`을 표시한다.
- 서비스를 한 줄로 소개하는 문구를 표시한다. (문구 미확정 — 상수로 분리하여 추후 교체 가능하게 한다)

### 시작 버튼

- 퀴즈 목록 화면으로 이동할 수 있는 시작 버튼을 제공한다.
- 버튼 클릭 시 `/quiz` (퀴즈 목록 선택 화면)으로 이동한다.

### 문서 링크 카드

- 학습에 유용한 공식 문서로 바로 이동할 수 있는 카드 목록을 제공한다.
- 각 카드는 **이름 + 한 줄 설명 + 외부 링크 아이콘(↗)** 으로 구성한다.
- 카드 목록은 상수 배열로 관리하여 항목 추가/제거가 용이하게 한다.
- 초기 제공 목록 (4개):

| 이름                     | URL                                 |
| ------------------------ | ----------------------------------- |
| MDN Web Docs             | https://developer.mozilla.org       |
| React 공식 문서          | https://react.dev                   |
| TypeScript 공식 문서     | https://www.typescriptlang.org/docs |
| JavaScript 모던 튜토리얼 | https://ko.javascript.info          |

## UI / UX 요구사항

- **레이아웃**: 모바일 퍼스트. 콘텐츠를 세로로 배치하는 앱 메인 화면 구조
- **컬러 테마**: 시스템 설정(`prefers-color-scheme`)을 따라 라이트/다크 모드 모두 대응
- **진입 애니메이션**: 페이지 진입 시 콘텐츠가 페이드인으로 나타난다
- **외부 링크**: 문서 카드 클릭 시 새 탭(`target="_blank"`, `rel="noreferrer"`)으로 열린다

## 비고

- 초기 버전에서는 로그인, 회원가입, 서버 연동을 제외한다.
- 홈 페이지는 정적 UI 중심으로 구현한다.
- 소개 문구 등 텍스트 콘텐츠는 상수로 분리하여 향후 수정이 쉽도록 한다.
- 문서 링크 카드 목록은 상수 배열로 관리한다.
